# Building a simple Ethereum DApp with Vue, uPort, Web3 and Ganache

Works with Node version 10 

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
